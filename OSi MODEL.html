What is OSI MODEL

OSI stands for open system interconnection model it has been developed by International standard organization (ISO) in 1984.
It is a 7 layers Architecture Where each layer specific funtionally. All these 7 layer work Collaborality to transmit the data from
one network to another network across the globe.


7 Layers Of OSI MODEL

01: Physical Layer

In This Layer Data is converted into electrical signals(0 and 1) or light pulses and sent over wires, fiber optics, or through wireless signals

02: Data Link Layer

This layer is responsible to transmit data using physical addresses. Data Link Layer ensures error free transmission of packets.

03: Network Layer

The function of this layer is the selection of the shortest and suitable path from source to destination from the number of routes available.

04: Transport Layer

This layer makes sure the data is broken into smaller pieces (so itâ€™s easier to send) and it ensures that all the pieces arrive correctly.

05: Session Layer

It opens, manages, and closes communication sessions between devices.

06: Presentation Layer

This layer translates data into a format the receiving device can understand. It also encrypts and compresses data if needed.

07: Application Layer

Where users and applications interact with the network.
